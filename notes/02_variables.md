# 02. 변수와 자료형

## 변수란?
값을 저장하는 메모리 공간.

## 기본 자료형(Primitive types)

| 자료형 | 설명 | 예 |
|-------|------|-----|
| int | 정수 | 10 |
| double | 실수 | 3.14 |
| char | 문자(작은 따옴표) | 'A' |
| boolean | 논리 | true/false |
| String | 문자열(객체) | "Hello" |

## 예제

```java
int age = 20;
double pi = 3.14;
char grade = 'A';
boolean pass = true;
String name = "Kim";
